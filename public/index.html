API Reference

All endpoints accept a URL parameter and return data with caching headers. Add your API key via X-API-Key header or ?api_key= query param.

GET /api/v1/favicon Fetch a site's favicon
ParameterTypeDescription
urlstringrequiredThe target URL
sizenumberoptionalResize to NxN pixels
formatstringoptionalOutput format: png, webp, jpeg (default: png)
# Get GitHub's favicon as a 64x64 PNG curl "https://api.opengrab.dev/api/v1/favicon?url=https://github.com&size=64" \ -H "X-API-Key: og_demo_key_12345" \ -o favicon.png
GET /api/v1/og Get Open Graph metadata (JSON)
ParameterTypeDescription
urlstringrequiredThe target URL
# Get OG metadata for a URL curl "https://api.opengrab.dev/api/v1/og?url=https://github.com" \ -H "X-API-Key: og_demo_key_12345" # Response: { "success": true, "data": { "title": "GitHub", "description": "Where the world builds software", "image": "https://github.githubassets.com/images/...", "siteName": "GitHub" } }
GET /api/v1/og/image Get OG image (auto-generates if missing)
ParameterTypeDescription
urlstringrequiredThe target URL

Returns the original OG image if the site has one. If not, generates a clean placeholder using the site's title and metadata.

GET /api/v1/meta Combined metadata + links
ParameterTypeDescription
urlstringrequiredThe target URL

Returns all metadata (OG, Twitter Card, favicon, theme color) plus convenient links to fetch images directly.

GET /api/v1/generate Generate custom OG images
ParameterTypeDescription
titlestringrequiredMain heading text
subtitlestringoptionalSubtitle text
domainstringoptionalDomain shown at bottom
bgstringoptionalBackground color (hex)
colorstringoptionalText color (hex)
accentstringoptionalAccent color (hex)
# Use in your HTML meta tags — images generated on the fly <meta property="og:image" content="https://api.opengrab.dev/api/v1/generate?title=My+Blog+Post&domain=myblog.com" />
